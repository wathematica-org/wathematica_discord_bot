* 「ゼミ長」とは
`/new`コマンドを使用した人が「ゼミ長」として登録される \
ゼミの削除(`/delete`)，ゼミ(休止中)への移行(`/pause`)はゼミ長しかできない \
`/chleader`コマンドでゼミ長変更ができる
